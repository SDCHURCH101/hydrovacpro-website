# Hydrovac Pro — Website

Marketing site for **Hydrovac Pro** (Sabe Capital), a hydro excavation and vacuum
truck company in Fairbanks, Alaska. Non-destructive daylighting, potholing, slot
trenching, Vactor 2100 jetter and culvert cleaning, tank and pit cleanouts,
frozen-ground excavation, and emergency response for utilities, government,
mining, oil and gas, construction, and environmental clients.

- **Email:** office@hydrovacpro.com

## Stack

Static HTML/CSS/JS, no build dependencies. The pages are generated by a single
Python script so shared header, footer, SEO, and structured data stay in sync.

```
python3 build.py
```

This regenerates `index.html`, `services.html`, `equipment.html`,
`industries.html`, `about.html`, `contact.html`, `404.html`, `robots.txt`,
`sitemap.xml`, and `site.webmanifest`. Edit `build.py` (and
`assets/styles.css` / `assets/app.js`), not the HTML files directly. Bump the
`VER` constant in `build.py` when you change the CSS or JS to cache-bust.

## Structure

- `build.py` — page generator (content, SEO, JSON-LD)
- `assets/styles.css`, `assets/app.js` — styles and front-end behavior
- `assets/img/` — logos, photos, favicons, social image
- `*.html` — generated pages (do not hand-edit)

## Hosting

Served by GitHub Pages from the repository root. The contact form delivers to
